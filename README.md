# LinuxDev-Installer

this script installs:
  + Git
  + Mono (.NET Framework)
  + Java
  + NodeJS + NPM
  + Snap (if you are not using Ubuntu)
  + PIP (Python2 and Python3)
  + Apache2
  + and much more
  
  ***How to install:***
```bash
sudo chmod +x install.sh && ./installer.sh

#now just wait
```
